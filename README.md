# LetsGoBiking
Full-stack Bike-focused Navigation App and Service

## Architecture

```
.
├── Client
├── LetsGoBiking.sln
├── README.md
├── Router
└── docker-compose.yml
```

## How to run
To run the backend services, first build the images:
```
docker compose build
```
then run them
```
docker compose up -d
```
